# Greasy Workspace — 油猴插件工作区

Tampermonkey userscript monorepo. Each `projects/<plugin>/` is independently buildable and publishable. See `README.md` for plugin list and `CONTEXT.md` (when present) for domain vocabulary.

## Agent skills

### Issue tracker

GitHub Issues on `bilbilmyc/Greasy`; use the `gh` CLI. External PRs are NOT a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles, default strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout; `CONTEXT.md` and `docs/adr/` (when present) live at the repo root. See `docs/agents/domain.md`.
