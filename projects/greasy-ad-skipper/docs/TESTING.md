# 🧪 内测灰测指南 — 实验 2：反检测有效性验证

## 测试目标

验证油猴脚本的反检测策略能否有效规避各平台的 adblock 检测，同时成功跳过广告。

## 测试者须知

### 安装方法

1. **确保已安装 Tampermonkey**
   - Chrome: Chrome 网上应用店搜索 "Tampermonkey"
   - Edge: Edge 扩展商店搜索 "Tampermonkey"
   - Firefox: 附加组件搜索 "Tampermonkey"

2. **安装脚本**
   - 打开 `dist/greasy-ad-skipper.user.js`（构建产物）
   - Tampermonkey 会自动弹出安装页面
   - 点击"安装"

3. **验证安装**
   - 浏览器右上角 Tampermonkey 图标 → 管理面板
   - 确认 "视频广告跳过助手" 在列表中，且开关为打开状态

### 测试内容

#### 第一部分：基本功能测试（必测）

| 平台 | 测试场景 | 预期结果 |
|------|---------|---------|
| **YouTube** | 打开任意视频（含广告） | 广告自动跳过，无黑屏/静音 |
| **YouTube** | 连续观看 5 个不同视频 | 每个都正常跳过 |
| **B站** | 打开任意视频（非大会员） | 贴片广告跳过 |
| **腾讯视频** | 打开任意剧集 | 片头广告跳过 |
| **爱奇艺** | 打开任意视频 | 广告跳过 |
| **优酷** | 打开任意视频 | 广告跳过 |
| **芒果TV** | 打开任意视频 | 广告跳过 |

#### 第二部分：反检测测试（核心）

| 测试项 | 操作 | 通过标准 |
|-------|------|---------|
| YouTube 检测 | 正常使用 3 天 | 没有出现 "Ad blocker detected" 弹窗 |
| YouTube 黑屏 | 观察视频播放 | 视频正常播放，无黑屏 |
| YouTube 静音 | 观察音频 | 视频有声音（不是被强制静音） |
| 多平台切换 | 在不同平台间切换 | 各平台都能正常跳过 |

#### 第三部分：配置面板测试（可选）

| 测试项 | 操作 |
|-------|------|
| 打开面板 | 点击悬浮球，配置面板应出现 |
| 切换模式 | 切换 全自动/隐身/仅标记，观察行为变化 |
| SponsorBlock | 开启/关闭 SponsorBlock，YouTube 行为应变化 |
| 悬浮球拖拽 | 尝试拖拽悬浮球到不同位置 |
| 统计显示 | 观察"今日跳过"计数是否正确 |

## 开发者：测试数据分析

### 成功标准

| 指标 | 目标 |
|------|------|
| 反检测通过率 | 7 天内 0 人触发 YouTube 检测警告 |
| 广告跳过成功率 | > 90% 的广告被成功跳过 |
| 误报率 | < 5%（非广告内容被误判为广告） |
| 配置面板使用率 | > 60% 的测试者至少打开过一次 |

### 决策框架

```
反检测 7 天零封禁 + 跳过率 > 90%
  → ✅ 继续推进，准备正式发布
反检测有封禁但可修复
  → 🟡 调整策略 → 再次灰测
反检测频繁封禁
  → ❌ 重新设计反检测架构，或考虑降低检测频率
```
